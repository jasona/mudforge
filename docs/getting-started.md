# Getting Started

## First Login

1. Create your character.
2. Run `help` for topic index.
3. Run `settings` to view and tune personal options.
4. Use `look`, `score`, and movement commands to explore.

## Useful Commands

- `help <topic>`
- `settings`
- `inventory` / `equipment`
- `consider <target>`
- `report <message>` (for abuse or harassment)

## Survival Basics

- Keep healing supplies ready.
- Set `wimpy` to a safe threshold.
- Optionally set `wimpycmd` to a safe command such as `flee` or `quaff`.
