# mudlib/cmds/senior/ - Senior Builder Commands (3 commands)

Permission level 2 - available to senior builders and above.

## Commands

- `_announce.ts` - System announcements (list, show, create via IDE, edit, delete)
- `_purge.ts` - Purge/cleanup operations
- `_reboot.ts` - Server reboot coordination

## Notes

Senior builders can write to `/lib/` and `/cmds/` directories in addition to `/areas/`.
